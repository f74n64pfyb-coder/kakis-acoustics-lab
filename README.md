# Kaki's Acoustics Lab

A modular acoustic design PWA.

## Included in v0.1
- New project workspace UI
- Existing Room Acoustics calculator preserved under `/room`
- Multi-Layer Absorber beta for porous + air layers
- Miki / transfer-matrix normal-incidence calculation
- 63–8000 Hz octave results and full response chart
- Local autosave, material library, dark mode, PWA shell

## GitHub Pages
Upload all files to the repository root, then enable Settings → Pages → Deploy from branch → main / root.
